void cylpa_olm_deinit ( olm_t *  olm)

Offload Manager de-init and free resources.

Parameters
olm: The pointer to the olm structure olm_t

Offload Manager de-init and free resources.

Initialize each offload in the offload list by calling each offload's initialization function (whd pointer is passed for IOVAR calls, and network interface for ip). If an offload initialization fails, stop and deinitialize any initialized offloads and return the result code of the failing initialization function.

Parameters
olmThe pointer to the olm structure olm_t.

◆ cylpa_olm_init_wlan_config()

void cylpa_olm_init_wlan_config ( void *  whd)

Configure wlan for low power using IOVAR(s) called once after cylpa_olm_init.

Parameters
whd: The pointer to the whd interface

Configure wlan for low power using IOVAR(s) called once after cylpa_olm_init.

This includes following steps:

  1. set the bcntrim iovar value ( LPA_BCNTRIM)
  2. set the bcn_wait_prd iovar value ( LPA_BCN_WAIT_PERIOD)
  3. set the bcn_reacquire_start iovar value ( LPA_BCN_REACQUIRE_START)
  4. set the roam_time_thresh iovar value ( LPA_ROAM_TIME_THRESHOLD )
Parameters
whdThe pointer to the whd interface.
