enum cy_awsport_rootca_verify_mode_t

Represents the RootCA verification mode for MQTT/HTTP client socket connection.

Enumerator
CY_AWS_ROOTCA_VERIFY_REQUIRED 

Peer RootCA verification to be done during TLS handshake.

Handshake is aborted if verification fails. This mode is a default verify mode for client sockets.

CY_AWS_ROOTCA_VERIFY_NONE 

Skip Peer RootCA verification during TLS handshake.

CY_AWS_ROOTCA_VERIFY_OPTIONAL 

Peer RootCA verification to be done during TLS handshake.

Even if the RootCA verification fails, continue with the TLS handshake.
