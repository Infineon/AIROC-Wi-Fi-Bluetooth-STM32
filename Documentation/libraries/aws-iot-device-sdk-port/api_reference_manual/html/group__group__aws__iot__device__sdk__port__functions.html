Sets up a TLS session and socket creation using the secure sockets API.

Note
This API and cy_awsport_network_connect API should always be called together. The secure sockets lwIP port layer has a limitation that it will delete the connection instance created in cy_awsport_network_create() as part of cy_awsport_network_disconnect().
Parameters
network_context[in] : Client network context.
server_info[in] : Server connection info.
ssl_credentials[in] : Credentials for the TLS connection.
discon_cb[in] : Socket disconnect callback function.
receive_cb[in] : Socket data receive callback function.
Returns
cy_rslt_t : CY_RSLT_SUCCESS on success; secure sockets library error code otherwise.

◆ cy_awsport_network_connect()

cy_rslt_t cy_awsport_network_connect ( NetworkContext_t *  network_context,
uint32_t  send_timeout,
uint32_t  recv_timeout 
)

Sets up a TCP connection using the secure sockets API.

Note
  1. This API should be called in pair with cy_awsport_network_create() API. The secure sockets lwIP port layer has a limitation that it will delete the connection instance created in cy_awsport_network_create() as part of the cy_awsport_network_disconnect() API. Therefore, cy_awsport_network_create() should be called before this API in pair.
  2. A timeout of 0 means infinite timeout.
Parameters
network_context[in] : Client network context.
send_timeout[in] : Timeout in milliseconds for transport send.
recv_timeout[in] : Timeout in milliseconds for transport recv.
Returns
cy_rslt_t : CY_RSLT_SUCCESS on success; secure sockets library error code otherwise.

◆ cy_awsport_network_disconnect()

cy_rslt_t cy_awsport_network_disconnect ( NetworkContext_t *  network_context)

Disconnects a TCP socket's connection using the secure sockets API.

Note
lwIP has a limitation: when disconnect is called, it will clear the netconn instance created and it will not allow any operation on that instance. Therefore, after disconnect, the connect API cannot be called directly. In addition, the secure socket must free its resources. Due to this lwIP limitation, this API and cy_awsport_network_delete() must be called for every disconnect.
Parameters
network_context[in] : Client network context.
Returns
cy_rslt_t : CY_RSLT_SUCCESS on success; secure sockets library error code otherwise.

Receives the data over an established TLS session using the secure sockets API.

This can be used as the transport-interface receive function for receiving data from the network.

Parameters
