Note
These strings must be NULL-terminated because the secure sockets API requires them to be. Memory allocated for sni_host_name, root_ca, client_cert, private_key, username, and password should be maintained by the caller until MQTT/HTTP object is deleted. Set to NULL to disable SNI.
