Anycloud Manufacturing Test
Overview

This application is the manufacturing test application for testing the Wlan firmware capabilities.This application is used along with wl tool to issue commands to Wlan Manufacturing firmware, getting response and displaying output.

Supported Platforms

This application can be run on the following Cypress platforms:

  • PSoC 6 Wi-Fi BT Prototyping Kit (CY8CPROTO-062-4343W)
  • PSoC 62S2 Wi-Fi BT Pioneer Kit (CY8CKIT-062S2-43012)

Dependent Libraries

This library includes Wi-Fi Middleware Core by default. It helps code examples to enable Wi-Fi connectivity components without having to include additional libraries. The Wi-Fi Host Driver library is part of the Wi-Fi Middleware Core.
